## Описание проекта

**Galaxy Strike** — это динамичный космический рельсовый шутер (Rail Shooter) с видом от третьего лица. Игрок управляет космическим истребителем, уничтожает волны врагов и маневрирует, уклоняясь от препятствий. В проекте реализована система прицеливания с использованием курсора мыши, подсчет очков и диалоговая система.

## Ключевые возможности

- **Рельсовый геймплей**: Корабль игрока движется по заданному маршруту, сохраняя свободу маневрирования в пределах экрана (X/Y clamp).
- **Боевая система**:
  - Управление стрельбой лазерами через Particle System.
  - Динамическое прицеливание: лазеры поворачиваются в точку, куда указывает перекрестие (Crosshair).
  - Система здоровья врагов и визуальные эффекты уничтожения (VFX).
- **Управление**:
  - Плавное перемещение корабля с наклонами (Roll/Pitch) при маневрах.
  - Прицеливание мышью с конвертацией экранных координат в мировые (`ScreenToWorldPoint`).
- **Интерфейс и системы**:
  - Подсчет очков в реальном времени (`Scoreboard`).
  - Диалоговая система с таймлайном (`DialogueLines`).
  - Синглтон для фоновой музыки (`MusicPlayer`) с сохранением между сценами (`DontDestroyOnLoad`).

## Техническая реализация

- **PlayerMove.cs**: Контроллер движения. Использует `Mathf.Clamp` для ограничения зоны полета и интерполяцию (`Quaternion.Lerp`) для плавных наклонов корпуса при поворотах.
- **PlayerWeapon.cs**: Система вооружения. Реализует механику стрельбы через управление эмиссией частиц и поворот эмиттеров в сторону точки прицеливания (`Quaternion.LookRotation`).
- **Enemy.cs**: Логика врагов. Обрабатывает попадания частиц (`OnParticleCollision`), уменьшает здоровье и начисляет очки при уничтожении.
- **GameSceneManager.cs**: Управление состоянием игры. Обрабатывает столкновения игрока и перезагружает уровень с задержкой (Coroutine).

## Установка и запуск

1. **Клонирование репозитория**:
   ```bash
   git clone https://github.com/lenderq/Galaxy-Strike.git
   ```
2. **Открытие проекта**:
   - Запустите Unity Hub.
   - Нажмите "Add" и выберите папку с клонированным репозиторием.
   - Рекомендуемая версия Unity: 6.3 LTS.

3. **Запуск игры**:
   - Откройте сцену `Level1` в папке `Assets/Scenes`.
   - Нажмите кнопку Play в редакторе.

| Действие | Ввод | Описание |
| :--- | :--- | :--- |
| **Перемещение** | `WASD` / `Стрелки` | Маневрирование кораблем |
| **Стрельба** | `Пробел` / `ЛКМ` | Активация лазеров |
| **Прицеливание** | `Мышь` | Управление перекрестием |
